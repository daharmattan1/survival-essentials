## Scope

**What This IS**:
- Markdown-based core knowledge you can read offline
- Practical takeaways for realistic scenarios
- Quick reference for emergencies
- Living document that can be updated
- Public domain image resources for identification and techniques

**What This IS NOT**:
- Comprehensive survival encyclopedia
- PDF collection repository (we link to sources)
- Prepper manifesto
- Military field manual library

---

## Realistic Scenarios Covered

1. **Multi-day power outage** (winter storm, grid failure)
2. **Stranded during travel** (car breakdown in remote area)
3. **Natural disaster evacuation** (wildfire, hurricane, flood)
4. **Supply chain disruption** (food/water shortages)
5. **Medical emergency without EMS** (remote location, delayed response)
6. **Communication blackout** (cell towers down, internet out)

---

## How to Use

### During Normal Times
- Read through once to familiarize yourself
- Practice 2-3 skills (fire starting, water purification)
- Assemble basic kit based on recommendations
- Download public domain resources (FM 21-76, plant images)

### During Emergency
- Quick reference for critical needs (water, fire, first aid)
- Step-by-step instructions for unfamiliar tasks
- Decision frameworks for prioritizing actions
- Offline access to all core knowledge
